# PC Build

---

## Описание проекта

---

**PC Build** - лендинг-страница для сайта с курсами по сборки компьютеров. 
Проект на **React, TypeScript, SCSS** с архитектурой FSD. 
**Макет взят из открытых источников и используется только в учебных целях.**

---

## Стек технологий
---

### Основной фреймворк и язык:

* React: Версия 19.2.0 (с React DOM для рендеринга, React Router DOM 7.9.6 для SPA-роутинга).
* TypeScript: Версия ~5.9.3 (для типизации, с типами @types/react и т.д.).
---

### Сборка и инструменты:

* Vite: Версия 7.2.4 (быстрая сборка и dev-сервер, с @vitejs/plugin-react).
* SCSS/Sass: Версия 1.94.2 (для стилей, с поддержкой модулей).
* ESLint: Версия 9.39.1 (линтинг, с плагинами для React hooks и refresh).
* Prettier: Версия 3.6.2 (форматирование кода).
---

### Дополнительные зависимости:

* Formik: 2.4.9 (формы).
* clsx: 2.1.1 (условные классы CSS).
* include-media: 2.0.0 (миксины для медиа-запросов в SCSS).
* react-svg: 16.3.0 (работа с SVG).
* reset-css: 5.0.2 (сброс стилей).

---


## Структура проекта

```
src/

```
